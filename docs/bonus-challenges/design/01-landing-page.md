## Goal

Transform the homepage into a professional, visually appealing marketing landing page.

This challenge tests your:

- **Design sense** - Can you make it look professional?
- **UI/UX skills** - Is it intuitive and user-friendly?
- **Frontend polish** - Attention to spacing, typography, colors
- **Responsive design** - Does it work beautifully on all devices?
- **SEO knowledge** - Hidden metadata challenges

We're looking for candidates who understand good design, not just programming.

---

## What to Build

### Hero Section

- Compelling headline and subheadline
- Clear value proposition
- Primary call-to-action button
- Supporting visual or illustration

### Features Section

- Highlight key benefits for publishers and sponsors
- Use icons or illustrations
- Clear, scannable layout

### How It Works

- Step-by-step explanation
- Visual indicators (numbered steps, icons)
- Keep it simple and clear

### Call-to-Action

- Final push to sign up or learn more
- Make the button prominent
- Consider adding social proof

## SEO Challenge

There are hidden TODO comments in the codebase related to SEO. Find them and implement proper:

- Page titles and meta descriptions
- Open Graph tags for social sharing
- Semantic HTML structure
- Heading hierarchy

## Requirements

- Responsive design (mobile, tablet, desktop)
- Clean, professional appearance
- Fast loading (optimize images if any)
- Accessible (proper contrast, semantic HTML)
